# Test Generator ルーブリック

## テストの網羅性（0.0-1.0）
- 正常系テストが含まれている
- 異常系テスト（無効入力、エッジケース）が含まれている
- 境界値テスト（空入力、ゼロ、最大値）が含まれている

## テストの品質（0.0-1.0）
- Arrange-Act-Assert パターンが使われている
- テスト名が何をテストしているか明確
- 各テストが独立している（外部状態に依存しない）

## 実行可能性（0.0-1.0）
- 全テストが実際に実行できる（import エラーなし）
- 全テストが PASS する
- 不必要なモックが使われていない
