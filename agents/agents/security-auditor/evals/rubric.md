# Security Auditor ルーブリック

## 脆弱性検出の正確性（0.0-1.0）
- 各脆弱性に正確な CWE 番号が付与されている
- 脆弱性の重要度分類（CRITICAL/HIGH/MEDIUM/LOW）が適切
- false positive（非脆弱性の誤検出）が少ない

## 修正提案の品質（0.0-1.0）
- 各脆弱性に具体的な修正方法が記載されている
- 修正方法がセキュリティのベストプラクティスに沿っている
- 修正方法が実装可能（コード例付き）

## 網羅性（0.0-1.0）
- CRITICAL/HIGH な脆弱性を全て検出している
- 複数ファイルにまたがる脆弱性を検出している
- OWASP Top 10 に該当する問題をカバーしている
