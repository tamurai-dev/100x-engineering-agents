# 100x Engineering Agents

AI エージェントを活用し、エンジニアリングの生産性を **100 倍** に引き上げるためのプロジェクトです。

---

## 概要

100x Engineering Agents は、ソフトウェア開発ライフサイクル全体を AI エージェントで加速するフレームワーク・ツールキットを提供します。

### 主な特徴

- **タスク自動分解** — 大きな開発タスクを実行可能なサブタスクへ自動的に分割
- **コード生成・レビュー** — AI エージェントによるコード生成、レビュー、リファクタリング
- **テスト自動化** — テストケース生成からE2Eテストの実行まで自動化
- **CI/CD 統合** — GitHub Actions 等と連携し、継続的なフィードバックループを実現
- **マルチエージェント協調** — 複数の AI エージェントが協調して複雑な開発タスクを遂行
